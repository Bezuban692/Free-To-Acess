## 🛠️ Installation & Setup

Follow these steps to quickly set up and run the project:

### 1️⃣ Clone the Repository

git clone https://github.com/your-username/your-repo-name.git

cd your-repo-name

---

### 2️⃣ Install Dependencies (Recommended)

npm install

---

### ⚡ Force Installation (If Errors Occur)

If you face dependency issues, use:

npm install --force

OR

npm install --legacy-peer-deps

---

### 🚀 Run the Project (Development Mode)

npm run dev

---

### 🌐 Production Build

npm run build

npm start

---

## ⚡ Quick Start (All-in-One Command)

For fast setup:

git clone https://github.com/your-username/your-repo-name.git && cd your-repo-name && npm install --force && npm run dev

---

## 🧩 Requirements

Make sure you have installed:

- Node.js (Latest Recommended)
- npm (comes with Node.js)

---

## 🛑 Troubleshooting

🔸 If node_modules issues:
rm -rf node_modules package-lock.json
npm install

🔸 Clear npm cache:
npm cache clean --force

🔸 Check Node version:
node -v

---

## 💡 Tips for Best Performance

- Use latest Node.js version  
- Close unnecessary background apps  
- Use SSD for faster performance  
- Run project on localhost for best speed   
npm install   # or pip install -r requirements.txt
npm start     # or python app.py
